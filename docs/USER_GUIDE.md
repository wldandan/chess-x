## 🎮 功能介绍

### 1. AI 对弈

**位置**: 主页 → 点击"开始对弈"

**功能特点**：
- 选择不同风格的 AI 对手（模仿世界冠军棋风）
- 选择时间控制（快棋、慢棋、子弹棋等）
- 实时对弈，AI 自动响应

**操作说明**：
1. 点击"开始对弈"按钮
2. 选择对手类型和时间控制
3. 点击棋子拖动到目标位置即可走棋
4. 使用控制按钮：悔棋、认输、提和棋

**快捷键**：
- 点击棋子 → 点击目标格子 = 走棋
- 拖动棋子到目标格子 = 走棋

---

### 2. 战术训练

**位置**: 导航菜单 → "战术训练"

**功能特点**：
- 16 种战术类型（捉双、牵制、串击等）
- 自适应难度调整
- 实时反馈和提示
- XP 和等级系统

**操作说明**：
1. 进入训练页面，选择"开始战术训练"
2. 查看题目提示（如"用马同时攻击两个目标"）
3. 在棋盘上走出正确答案
4. 查看反馈，正确则进入下一题

**战术类型**：
| 图标 | 名称 | 描述 |
|------|------|------|
| ⚔️ | 捉双 | 用一个棋子同时攻击两个目标 |
| 📌 | 牵制 | 牵制对手棋子无法移动 |
| 🔱 | 串击 | 串击战术 |
| ⚡ | 闪击 | 闪将战术 |
| 🎯 | 双重攻击 | 同时攻击多个目标 |

**提示功能**：
- 卡住时点击"提示"按钮获取线索
- 可以"跳过"当前题目
- 可以"查看解答"学习正确走法

---

### 3. 对局分析

**位置**: 导航菜单 → "对局分析"

**功能特点**：
- AI 深度分析每一步棋
- 棋步评分（最佳、好棋、失误、大漏）
- 替代走法建议
- 战术机会检测
- 完整对局报告

**操作说明**：
1. 从对局历史中选择要分析的对局
2. 点击"开始分析"
3. 等待 AI 分析完成（约 10-30 秒）
4. 逐步复盘，查看每步的评分和建议

**棋步质量说明**：
| 颜色 | 评级 | 描述 |
|------|------|------|
| 🟢 绿色 | 最佳 | 完美的走法 |
| 🔵 蓝色 | 优秀 | 很好的走法 |
| ⚪ 灰色 | 正常 | 合理的走法 |
| 🟡 黄色 | 不精确 | 稍有偏差 |
| 🟠 橙色 | 失误 | 明显错误 |
| 🔴 红色 | 大漏 | 严重错误 |

**分析报告包含**：
- 整体准确率
- 白方/黑方准确率
- 错失的战术机会
- 开局分析
- 时间管理评估

## 🎨 界面说明

### 导航栏

```
┌─────────────────────────────────────────┐
│  Chess-X  首页  分析  训练  设置  用户  │
└─────────────────────────────────────────┘
```

- **首页** - 对弈入口和快捷功能
- **分析** - 对局分析和复盘
- **训练** - 战术训练和题目练习
- **设置** - 应用设置和偏好
- **用户** - 个人信息和统计数据

### 棋盘控制

```
┌─────────────────────────────────────┐
│  ⏮  ◀  ▶  ⏭    [逐步] [错误] [关键]  │
│  棋盘方向: ⚪                        │
└─────────────────────────────────────┘
```

- **⏮⏭** - 跳到开始/结束
- **◀▶** - 上一步/下一步
- **逐步/错误/关键** - 切换复盘模式
- **⚪⚫** - 切换棋盘视角

## ❓ 常见问题

### Q: 如何悔棋？

**A**: 在对弈页面点击"悔棋"按钮，可以撤回最后一步棋（包括你的和 AI 的）。

### Q: 为什么 AI 没有响应？

**A**:
- 检查网络连接
- 确认走法是否合法
- 等待几秒，AI 可能在思考

### Q: 如何提高战术训练等级？

**A**:
- 连续正确答题会自动提升难度
- 每解决一道题获得 XP
- 积累足够 XP 自动升级

### Q: 分析报告可以导出吗？

**A**: 可以，在分析页面点击"导出"按钮，选择 PGN 或 JSON 格式。

### Q: 数据会保存吗？

**A**:
- 对局历史自动保存在本地
- 训练进度自动保存
- 清除浏览器数据会丢失本地记录

### Q: 可以和真人下棋吗？

**A**: 当前版本仅支持 AI 对弈，多人对战功能正在开发中。

## 💡 使用建议

1. **新手建议**：
   - 从简单战术开始训练
   - 选择较低难度的 AI
   - 仔细阅读分析报告

2. **进阶建议**：
   - 定期复盘对局
   - 专注于弱项战术类型
   - 尝试不同时间控制

3. **训练建议**：
   - 每天至少完成 10 道战术题
   - 坚持复盘每局对弈
   - 记录自己的进步
